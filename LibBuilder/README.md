# Library Building

## Build RTOS

1. Change directory to `LibBuild/rtos`.
2. Edit the configuration file, `libbuild.cfg` if required.
3. Run a command `ecc-pic24-cli -library`.

## Build BSP
1. Change directory to `LibBuild/bsp`.
2. Edit the configuration file, `libbuild.cfg` if required.
3. Run a command `ecc-pic24-cli -library`.
